# ⚔️ Val's Claude Code Plugins

Personal Claude Code skills and plugins.

## Plugins

### `val-personal`
My full personal productivity toolkit — quest log + ADHD executive function modes.
```bash
git clone https://github.com/valcarabello-tech/claude-plugins ~/claude-plugins
claude plugin install ~/claude-plugins/plugins/val-personal
```

### `quest-log` (shareable)
Standalone quest log skill — DnD/LARP-themed weekly to-do list with XP leveling and a native macOS app. Installable from the `quest-log` branch.
```bash
git clone -b quest-log https://github.com/valcarabello-tech/claude-plugins ~/quest-log
claude plugin install ~/quest-log
```
